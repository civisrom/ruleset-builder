## 🚀 Быстрый старт

### 1. Запуск программы

**GUI режим** (графический интерфейс):
```bash
python ruleset_builder.py
```

**CLI режим** (командная строка):
```bash
python ruleset_builder.py -o output -f json --domain data/domains.txt
```

### 2. Первые шаги

1. Откройте программу
2. Выберите sing-box.exe (опционально)
3. Заполните правила или загрузите шаблон
4. Нажмите "Генерировать Ruleset"
5. Готово!

## 📝 Примеры использования

### Пример 1: Блокировка рекламы
```bash
# GUI: Выберите шаблон "Блокировка рекламы"
# CLI:
python ruleset_builder.py -o ads_block -f json \
  --domain-suffix data/suffixes.txt \
  --domain-keyword "ads,tracking,telemetry"
```

### Пример 2: VPN обход
```bash
python ruleset_builder.py -o vpn_bypass -f mrs \
  --domain "telegram.org,twitter.com" \
  --ip-cidr "149.154.160.0/20"
```

### Пример 3: Конвертация из Clash
```bash
python ruleset_utils.py convert \
  --from clash \
  --input clash_config.yaml \
  --output singbox.json
```

### Пример 4: Анализ ruleset
```bash
python ruleset_utils.py analyze \
  --input my_rules.json \
  --find-duplicates
```

## 🐛 Часто задаваемые вопросы

**Q: Нужны ли внешние зависимости?**  
A: Нет! Всё работает на чистом Python 3.7+

**Q: Как работать с большими файлами?**  
A: Программа оптимизирована для файлов до 1 GB

**Q: Поддерживается ли IPv6?**  
A: Пока нет, планируется в v3.1

**Q: Можно ли использовать без GUI?**  
A: Да, есть полноценный CLI режим

**Q: Как создать .exe файл?**  
A: Используйте PyInstaller или GitHub Actions
